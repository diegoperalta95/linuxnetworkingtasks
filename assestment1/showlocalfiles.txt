LS - shows the current files/directories in a given directory

ls -l - the l flag makes the command to show every on a list
ls -a - shows everything including the hidden files/directories
ls -t - sorts by time & date
ls -r - list in reverse order

A user can distinguish files from directories with the first character that appears at the start of every line, if its a directory a letter "d" will appear first, if it is a file a "-" will appear
